## Commands:

* `>admin` - to get admin perms.
* `>massunban` - to unban all banned users.
* `>massban` - to ban everyone on the server.
* `>massdeleteroles` - to mass delete all roles.
* `>massdeletechannels` - to mass delete all channels.
